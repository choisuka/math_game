🔢 수와 정수론

소수(素數)는 왜 무한히 많은가?

🔍 탐구하기

소수를 나열하면: 2, 3, 5, 7, 11, 13, …
소수가 언젠가 끝날까? 아니면 무한히 존재할까?

유클리드의 증명 (기원전 300년)

1
귀류법 가정: 소수가 유한 개 p₁, p₂, …, pₙ 뿐이라고 가정하자.
→ "아니라면"을 가정해 모순을 찾는 기법
2
N = p₁×p₂×…×pₙ + 1 을 만들자.
→ 모든 소수의 곱에 1을 더한 수
3
N을 어떤 소수 pᵢ 로 나눠도 나머지 = 1
→ pᵢ는 (p₁×…×pₙ)를 나누므로, N ÷ pᵢ의 나머지는 1
4
N은 소수이거나, 기존 목록에 없는 새 소수를 약수로 가짐 → 모순!
→ "소수가 유한 개"라는 가정이 틀렸음

∴ 정리: 소수는 무한히 많다.

이 증명은 귀류법(背理法)의 고전. 가정이 틀렸음을 보임으로써 결론을 확정한다.

✏️ 확인 문제

N = 2×3×5 + 1 = 31. 31은 소수인가? 2, 3, 5로 나눠지는지 확인해 보자.

31 ÷ 2 = 나머지 1, 31 ÷ 3 = 나머지 1, 31 ÷ 5 = 나머지 1. 31은 소수다. (항상 소수인 건 아님 — N = 2×3×5×7×11×13+1 = 30031 = 59×509)

🔢 수와 정수론

최대공약수를 왜 이렇게 구하나? — 유클리드 호제법

🔍 탐구하기

48과 18의 최대공약수(GCD)를 구하라.
소인수분해 없이 더 빠른 방법이 있을까?

핵심 원리 (왜 성립하나?) a = bq + r 이면 gcd(a, b) = gcd(b, r)
∵ a와 b의 공약수는 모두 r = a - bq 의 약수이기도 하고, 역도 성립

48과 18의 GCD 계산

1
48 = 18 × 2 + 12 → gcd(48,18) = gcd(18,12)
2
18 = 12 × 1 + 6 → gcd(18,12) = gcd(12,6)
3
12 = 6 × 2 + 0 → gcd(12,6) = 6
gcd(48, 18) = 6

∴ 알고리즘: 나머지가 0이 될 때까지 반복 나눗셈

매 단계마다 수가 줄어드므로 반드시 종료됨. 큰 수에서도 매우 빠르다.

✏️ 확인 문제

유클리드 호제법으로 gcd(84, 36)을 구하라.

84 = 36×2+12 → 36 = 12×3+0 → gcd = 12

🔢 수와 정수론

홀수×홀수는 왜 항상 홀수인가?

🔍 탐구하기

3×5=15(홀), 7×9=63(홀). 몇 가지 예시를 봤지만…
모든 홀수의 곱이 홀수임을 어떻게 확실히 증명할까?

대수적 증명

1
홀수를 일반적으로 표현: 2k+1 (k는 정수)
→ 2로 나눈 나머지가 1인 수
2
두 홀수 a = 2m+1, b = 2n+1의 곱:
a×b = (2m+1)(2n+1) = 4mn + 2m + 2n + 1
3
= 2(2mn + m + n) + 1 → 2×(정수) + 1 꼴
→ 이 형태는 항상 홀수의 정의를 만족

∴ 홀수 × 홀수 = 홀수 (항상 참)

예시가 아무리 많아도 증명이 아니다. 일반화된 대수 표현을 통해서만 "모든 경우"를 확정할 수 있다.

같은 방법으로: 짝수 × (어떤 수) = 짝수 (증명: 2m × n = 2(mn))
홀수 + 홀수 = 짝수 (증명: (2m+1)+(2n+1) = 2(m+n+1))

🔢 수와 정수론

√2는 왜 분수로 쓸 수 없나?

🔍 탐구하기

1.4142…로 계산기에서 보이는 √2.
혹시 p/q (정수의 비) 형태로 정확히 쓸 수 있지 않을까?

귀류법으로 증명

1
가정: √2 = p/q (기약분수, 즉 gcd(p,q)=1)
→ 만약 유리수라면 최소 형태의 분수로 표현 가능해야 함
2
양변 제곱: 2 = p²/q² → p² = 2q²
→ p²이 짝수 → p도 짝수 (홀수의 제곱은 홀수이므로)
3
p = 2k 로 놓으면: (2k)² = 2q² → 4k² = 2q² → q² = 2k²
→ q²도 짝수 → q도 짝수
4
p도 짝수, q도 짝수 → gcd(p,q) ≥ 2 → 기약분수 가정 모순!

∴ √2는 유리수가 아니다 — 무리수

피타고라스학파가 이 사실을 발견하고 충격을 받았다는 기록이 있다.
증명은 수학의 힘: 유한한 논리 단계로 무한한 소수점을 가진 수의 본질을 확정한다.

🎲 조합과 확률

곱의 법칙은 왜 성립하나?

🔍 탐구하기

상의 3종류, 하의 4종류. 총 조합 수는 3×4=12.
왜 더하는 것이 아니라 곱하는가?

왜 곱셈인가? 상의 하나를 고를 때마다 하의 선택지 4개가 독립적으로 반복된다.
3번 반복 × 4가지 = 12. 덧셈은 "또는(or)", 곱셈은 "그리고(and)".

✏️ 확인 문제

동전을 3번 던질 때 나올 수 있는 경우의 수는?

매번 2가지(앞/뒤) × 3번 = 2×2×2 = 8가지. (HHH, HHT, HTH, HTT, THH, THT, TTH, TTT)

🎲 조합과 확률

조합 공식 C(n,k) = n! / (k!(n-k)!) — 왜?

🔍 탐구하기

5명 중 3명을 뽑는 조합. 공식은 C(5,3) = 10.
이 복잡한 분수가 어떻게 "뽑는 방법 수"가 되는가?

단계별 논증

1
순서 있게 3명 뽑기(순열): 5×4×3 = 60가지
→ 첫 번째 5명 중, 두 번째 4명 중, 세 번째 3명 중 선택
2
같은 3명을 나열하는 방법: 3! = 6가지
→ {A,B,C}를 순서 있게 나열: ABC, ACB, BAC, BCA, CAB, CBA
3
순서 없이 선택 = 60 ÷ 6 = 10
→ 같은 조합을 6번 중복 세었으므로 나눠줌
C(n,k) = n×(n-1)×…×(n-k+1) / k! = n! / (k!(n-k)!)
핵심: 순열에서 "순서 차이를 제거"하면 조합이 된다.
순서 있는 선택(순열) ÷ 나열 방법 수(k!) = 순서 없는 선택(조합)

✏️ 확인 문제

C(6,2)를 공식으로 구하고, 직접 나열해서 검증하라.

C(6,2) = 6!/(2!×4!) = (6×5)/(2×1) = 15. 직접 나열: {1,2},{1,3},{1,4},{1,5},{1,6},{2,3},{2,4},{2,5},{2,6},{3,4},{3,5},{3,6},{4,5},{4,6},{5,6} = 15개 ✓

🎲 조합과 확률

기댓값 — 평균은 왜 이렇게 계산하나?

🔍 탐구하기

주사위를 한 번 던질 때 나오는 눈의 기댓값은?
(1+2+3+4+5+6)÷6 = 3.5. 이것이 왜 "기대되는 값"인가?

기댓값의 의미

1
주사위를 600번 던지면: 각 눈이 약 100번씩 나온다(균등 확률)
→ 확률 1/6 × 600번 = 100번
2
합계 = 1×100 + 2×100 + 3×100 + 4×100 + 5×100 + 6×100 = 2100
3
평균 = 2100 ÷ 600 = 1×(1/6) + 2×(1/6) + … + 6×(1/6) = 3.5
→ 각 값 × 확률의 합 = 기댓값의 정의
E(X) = x₁p₁ + x₂p₂ + … + xₙpₙ = Σ xᵢpᵢ
실생활 응용: 복권 기댓값
1000원짜리 복권, 1등(100만원) 확률 1/10000, 나머지는 꽝.
E = 100만×(1/10000) + 0×(9999/10000) = 100원 → 1000원 내고 100원 기대 → 손해

✏️ 확인 문제

동전을 던져 앞면=+200원, 뒷면=-100원이라면 기댓값은?

E = 200×(1/2) + (-100)×(1/2) = 100 - 50 = 50원. 이 게임은 장기적으로 유리하다.

🎲 조합과 확률

독립사건: P(A∩B) = P(A)×P(B) — 왜?

🔍 탐구하기

동전과 주사위를 동시에 던질 때 "앞면이고 6이 나올" 확률은?
1/2 × 1/6 = 1/12. 왜 곱하는가?

표본공간으로 증명

1
전체 경우: (앞/뒤) × (1~6) = 12가지
2
"앞면이고 6" 인 경우: (앞,6) = 1가지
3
P = 1/12 = (1/2)×(1/6) = P(앞)×P(6)
→ 두 사건이 서로 영향을 주지 않으면(독립) 곱셈으로 계산
독립이 아닌 경우 (주의!):
주머니에서 빨간 공 3, 파란 공 2. 하나 꺼낸 후 돌려놓지 않고 또 꺼낼 때
→ 첫 번째 결과가 두 번째 확률에 영향 → 독립이 아님 → 단순 곱 불가

✏️ 확인 문제

동전을 2번 던져 둘 다 앞면일 확률은?

P(앞)×P(앞) = 1/2 × 1/2 = 1/4. 각 던지기는 독립이므로 곱셈 사용.

📐 기하 증명

피타고라스 정리 — 넓이로 증명하기

🔍 탐구하기

직각삼각형에서 a²+b²=c².
이것을 눈으로 볼 수 있게 증명할 수 있을까?

넓이로 증명

1
큰 정사각형 넓이 = (a+b)² = a²+2ab+b²
2
= 안쪽 정사각형 c² + 4×(직각삼각형) = c² + 4×(ab/2) = c²+2ab
3
a²+2ab+b² = c²+2ab → a²+b² = c²

∴ 피타고라스 정리 증명 완료

넓이 보존 원리를 이용한 시각적이고 우아한 증명. 2,000년 이상 수학자들이 애용해온 방법.

📐 기하 증명

원주각은 왜 중심각의 절반인가?

🔍 탐구하기

원 위의 같은 호에 대한 원주각들은 모두 같고, 중심각의 절반이다.
왜 그럴까?

이등변삼각형을 이용한 증명 (핵심 아이디어)

1
OA = OB = OP = 반지름 → △OAP, △OBP 는 이등변삼각형
2
이등변삼각형의 외각 정리: ∠AOQ = 2∠APO, ∠BOQ = 2∠BPO
→ 외각 = 내각 두 개의 합, 이등변삼각형이면 두 내각이 같음
3
중심각 ∠AOB = ∠AOQ + ∠BOQ = 2(∠APO + ∠BPO) = 2 × 원주각

∴ 원주각 = 중심각의 1/2

같은 호에 대한 원주각은 위치에 무관하게 모두 동일 → 반원에 대한 원주각 = 90°

📐 기하 증명

삼각형 넓이 공식 — 왜 (1/2)ab sinC ?

🔍 탐구하기

직각삼각형이 아닌 일반 삼각형의 넓이는?
넓이 = (1/2)×밑변×높이 — 이 높이를 sinC로 표현할 수 있다.

높이를 sinC로 표현

1
꼭짓점 C에서 밑변 AB에 수선을 내리면 높이 h 생성
2
직각삼각형에서: sin A = h/b → h = b·sinA
→ sin = 높이/빗변 의 정의
3
넓이 = (1/2)×c×h = (1/2)×c×b·sinA = (1/2)bc sinA
S = (1/2)ab sinC = (1/2)bc sinA = (1/2)ca sinB
의미: 두 변과 그 사이의 각도만 알면 삼각형 넓이를 구할 수 있다.
직각삼각형 공식 S=(1/2)×밑×높이 의 일반화된 버전.

📐 기하 증명

코사인 법칙 — 피타고라스의 일반화

🔍 탐구하기

직각삼각형 아닌 경우 a²+b²=c²은 틀린다.
c² = a² + b² - 2ab cosC 가 일반적 공식인데, 왜?

좌표 이용 증명

1
A를 원점, B를 (c, 0)에 놓으면:
C의 좌표 = (b cosA, b sinA)
→ 극좌표 표현: A에서 거리 b, 각도 A 방향
2
a² = BC² = (c - b cosA)² + (b sinA)²
3
전개: c² - 2bc cosA + b²cos²A + b²sin²A
= c² - 2bc cosA + b²(cos²A + sin²A)
= c² - 2bc cosA + b² (∵ sin²+cos²=1)
a² = b² + c² - 2bc cosA
C=90° 이면: cosC = 0 → c² = a² + b² → 피타고라스 정리!
코사인 법칙은 피타고라스의 완전한 일반화다.

✏️ 확인 문제

두 변 a=5, b=7, 끼인각 C=60°일 때 c를 구하라.

c² = 5² + 7² - 2×5×7×cos60° = 25 + 49 - 70×(1/2) = 74 - 35 = 39
c = √39 ≈ 6.24

📈 함수 분석

이차함수 꼭짓점 공식은 어떻게 나오나?

🔍 탐구하기

y = ax² + bx + c의 꼭짓점의 x좌표가 -b/(2a)라고 배웠다.
왜 이 값이 꼭짓점인가?

완전제곱식으로 변환 (완전제곱식 완성)

1
y = ax² + bx + c = a(x² + (b/a)x) + c
2
x² + (b/a)x = [x + b/(2a)]² - b²/(4a²)
→ (x+p)² = x²+2px+p²이므로 p = b/(2a)
3
y = a[x + b/(2a)]² - b²/(4a) + c
→ a가 양수면 [...]² ≥ 0이므로 x = -b/(2a) 일 때 최솟값
꼭짓점: x = -b/(2a), y = c - b²/(4a)
의미: 이차함수를 [x의 이동]² 꼴로 바꾸면 꼭짓점이 자동으로 보인다.
완전제곱식은 함수의 대칭축과 최솟값/최댓값의 위치를 한눈에 보여준다.

✏️ 확인 문제

y = x² - 4x + 7의 꼭짓점을 완전제곱식으로 구하라.

y = (x²-4x+4) + 3 = (x-2)² + 3. 꼭짓점: (2, 3). 공식 확인: -b/2a = 4/2 = 2 ✓

📈 함수 분석

근의 공식 — 왜 x = (-b ± √(b²-4ac)) / 2a ?

🔍 탐구하기

ax²+bx+c=0의 해를 항상 구할 수 있다는 근의 공식.
이 복잡한 식이 어떻게 유도되는가?

완전제곱식으로 유도

1
ax²+bx+c = 0 → a로 나누기: x² + (b/a)x + c/a = 0
2
완전제곱식: [x + b/(2a)]² = b²/(4a²) - c/a
→ 앞에서 배운 완전제곱식 기법 적용
3
우변 통분: (b²-4ac)/(4a²)
4
x + b/(2a) = ±√(b²-4ac)/(2a) → x = (-b ± √(b²-4ac)) / 2a
판별식 D = b²-4ac: D>0 → 두 실근, D=0 → 중근, D<0 → 실근 없음
판별식의 의미: √ 안이 음수이면 실수 해가 없다.
수학적으로는 허수(복소수) 해가 존재하지만, 중학교 범위에서는 실근 여부 판별에 활용.

📈 함수 분석

지수와 로그 — 서로의 역함수임을 확인

🔍 탐구하기

2³ = 8 이면, 8을 2의 몇 제곱으로 써야 하는가?
log₂8 = 3. 로그는 지수의 역사고다.

지수와 로그의 관계

1
지수: a^x = y → "a를 x번 곱하면 y가 된다"
→ x가 입력, y가 출력
2
로그: log_a(y) = x → "a를 몇 번 곱해야 y가 되나?"
→ y가 입력, x가 출력 (지수와 입출력이 뒤바뀜)
3
따라서 f(x) = a^x 와 g(x) = log_a(x) 는 y=x에 대해 대칭인 역함수
지수식로그식값
2³ = 8log₂8 = 38
10² = 100log₁₀100 = 2100
3⁴ = 81log₃81 = 481
5¹ = 5log₅5 = 15
실생활 응용: 은행 복리 계산
원금이 2배가 되려면? 2 = (1.05)^n → n = log₁.₀₅(2) ≈ 14.2년
→ 5% 복리라면 약 14년 후 원금이 두 배

📈 함수 분석

등차수열의 합 공식 — 가우스의 방법

🔍 탐구하기

1+2+3+…+100 = ?
10살의 가우스가 5초 만에 풀었다는 유명한 이야기.

가우스의 아이디어

1
S = 1 + 2 + 3 + … + 100
S = 100 + 99 + 98 + … + 1 (역순으로 한 번 더 쓰기)
2
두 줄 더하기: 2S = (1+100)+(2+99)+(3+98)+… = 101 × 100
→ 각 쌍의 합이 모두 101, 총 100쌍
3
S = 101 × 100 / 2 = 5050
S_n = n(첫째항 + 끝항)/2 = n(a₁ + aₙ)/2
일반화: 등차수열 a₁, a₂, …, aₙ (공차 d)
S_n = n×a₁ + d×n(n-1)/2
→ 이 공식도 같은 "역순 더하기" 방법으로 유도 가능

✏️ 확인 문제

3+7+11+…+43의 합은? (등차수열, 공차 4)

항 수: (43-3)/4 + 1 = 11항. S = 11×(3+43)/2 = 11×23 = 253

표준편차 — 왜 분산의 제곱근을 쓰나?

🔍 탐구하기

두 반의 수학 점수 평균이 70점으로 같다. 어떤 반이 더 고르게 잘 봤나?
평균만으로는 알 수 없다 — 퍼짐 정도가 필요하다.

표준편차 계산 과정

1
각 값과 평균의 차(편차): (xᵢ - x̄)
→ 편차의 합은 항상 0 (양수·음수 상쇄)
2
편차를 제곱해서 합산 → 분산: σ² = Σ(xᵢ-x̄)² / n
→ 제곱으로 부호 제거, 큰 편차에 더 큰 가중치
3
제곱근: σ = √(분산)
→ 단위를 원래 데이터와 같게 복원 (점²→점)

예시: 두 반의 점수

A반
68, 70, 70, 72 (σ≈1.4)
B반
50, 60, 80, 90 (σ≈15.8)

둘 다 평균 70점. A반이 훨씬 고른 점수 분포.

∴ 표준편차 = 데이터가 평균에서 얼마나 퍼져 있나?

제곱→제곱근의 이유: 음수를 없애면서 단위를 보존하는 가장 자연스러운 방법.

📊 통계 추론

상관관계 ≠ 인과관계 — 통계의 함정

🔍 탐구하기

"아이스크림 판매량이 늘면 익사 사고도 늘어난다" → 아이스크림이 익사를 유발하는가?

상관관계와 인과관계의 차이

1
상관관계: 두 변수가 함께 증가/감소하는 경향
→ 아이스크림↑ 동시에 익사↑ (관찰)
2
숨겨진 변수: 여름 더위가 원인
→ 더울 때 아이스크림도 많이 팔리고, 수영도 많이 함
3
인과관계 주장을 위해서는 통제된 실험(대조군)이 필요
⚠️ 잘못된 추론 사례

· 소방관 수 ↑ → 화재 피해 ↑ (원인: 큰 화재일수록 소방관이 많이 투입)
· 키 큰 사람이 성적이 높다 (원인: 나이가 많을수록 키도 크고 공부도 더 함)
· 교회 수 ↑ → 범죄율 ↑ (원인: 도시 규모 — 인구가 많으면 둘 다 많음)

핀란드 통계 교육의 핵심:
숫자를 믿기 전에 "왜?"를 물어라. 상관관계가 있어도 원인과 결과를 직접 연결하려면 추가 증거가 필요하다.

🏆 핵심 정리

심화과정 전체 요약 — "왜를 확정하는 5가지 무기"

1. 귀류법 (수론·무리수)

"아니다"를 가정해 모순을 찾아 "맞다"를 확정. 소수의 무한성, √2의 무리수성 증명.

2. 일반화 (대수적 증명)

특정 예시가 아닌 문자(2k+1)로 표현해 "모든 경우"를 한 번에 커버. 홀수×홀수=홀수 증명.

3. 넓이 보존 (기하 증명)

같은 도형을 두 가지 방법으로 계산해 등식 유도. 피타고라스 정리, 삼각형 넓이 공식 유도.

4. 역함수 관계 (함수 분석)

입출력을 뒤집으면 역함수. 지수↔로그, 완전제곱식 완성으로 꼭짓점·근의 공식 유도.

5. 통제 실험 (통계 추론)

상관관계를 인과관계로 착각하지 말 것. 숨겨진 변수를 찾고 대조군으로 검증해야 "왜"가 확정된다.

🌲 핀란드 수학 철학

"Matematiikka on ymmärtämistä, ei muistamista."
(수학은 기억이 아니라 이해다.)

공식을 외우는 것이 아니라, 그 공식이 왜 참인지 스스로 납득하는 것 — 이것이 핀란드 심화과정의 핵심이다.
