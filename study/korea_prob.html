📌 대표 문제

연속하는 세 홀수가 있다. 가장 큰 홀수와 가장 작은 홀수의 곱이 중간 홀수의 9배보다 4 크다고 할 때, 세 홀수를 구하여라.

🇫🇮 핀란드식 풀이

설정

중간 홀수를 x라 하면: 세 홀수는 x-2, x, x+2

홀수의 간격은 항상 2 → 중간을 x로 놓으면 양쪽이 x±2로 대칭! 식이 더 간단해집니다.
방정식
(x-2)(x+2) = 9x + 4
가장 큰×가장 작은 = 중간의 9배+4. 좌변은 합차공식 a²-b²!
풀기
x² - 4 = 9x + 4 → x² - 9x - 8 = 0
(x-1)(x-8) = 0? → 인수분해 안 됨 → 근의 공식 시도

잠깐 — 홀수 조건을 먼저 활용!

x² - 9x - 8 = 0 … (1)

판별식 D = 81 + 32 = 113 → 정수해 없음 ← 변수 설정 재검토

⚠ 막혔을 때: 변수를 바꿔서 재설정해 봅니다 (핀란드식 — 틀림은 배움의 일부!)
재설정

가장 작은 홀수를 x로: x, x+2, x+4

x(x+4) = 9(x+2) + 4
x² + 4x = 9x + 22
x² - 5x - 22 = 0 → (x-11)(x+2) = 0
답

x = 11 또는 x = -2 → 홀수이므로 x=11

세 홀수: 11, 13, 15
✓ 검산: 11×15 = 165 = 9×13 + 4 = 117+4 ✗ → 오류!
수정

문제 재독: "가장 큰 × 가장 작은 = 9×중간 + 4"

11 × 15 = 165,  9 × 13 + 4 = 121 ✗

올바른 설정: x-2, x, x+2로 돌아가서

x² - 4 = 9x + 4 → x² - 9x - 8 = 0

→ 이 문제는 정수해가 없습니다.

핀란드식 교훈: 검산 단계에서 오류를 잡는 것 자체가 수학 실력입니다! 조건이 맞지 않는 문제라면 문제 자체를 의심하세요.

📌 대표 문제 2 (정수해 있는 버전)

연속하는 세 홀수의 합이 57이고, 가장 큰 수와 가장 작은 수의 곱이 중간 수의 제곱보다 4 작다. 세 홀수를 구하여라.

🇫🇮 핀란드식 풀이

1

중간 홀수 = x로 놓기: x-2, x, x+2

조건①: (x-2) + x + (x+2) = 57 → 3x = 57 → x = 19
2

세 홀수: 17, 19, 21

조건② 검산: 17×21 = 357 = 19² - 4 = 361 - 4 ✓
세 홀수: 17, 19, 21
나이 문제
현재와 과거/미래의 나이 관계 — 표로 정리하면 보인다

🔑 핵심 전략 — 나이 표(table)로 현재·미래를 정리

나이 문제는 현재 나이를 x로 놓고, n년 후는 x+n, n년 전은 x-n으로 쓴 다음 표를 만들면 관계가 명확해집니다.

📌 대표 문제

현재 아버지의 나이는 아들 나이의 4배이다. 12년 후에는 아버지 나이가 아들 나이의 2배보다 6살이 많아진다. 현재 아버지와 아들의 나이를 구하여라.

🇫🇮 핀란드식 풀이 — 표 + 저울 모델

표
현재12년 후
아들xx + 12
아버지4x4x + 12
표를 먼저 그리면 "무엇이 변하고 무엇이 고정인지" 한눈에 보입니다. 나이 차이(3x)는 절대 변하지 않는다는 것도 확인!
방정식

12년 후 조건: 아버지 = 2 × 아들 + 6

4x + 12
=
2(x+12) + 6
4x + 12 = 2x + 24 + 6 = 2x + 30
2x = 18 → x = 9
답
현재 아들: 9세, 아버지: 36세
✓ 검산: 12년 후 → 아들 21세, 아버지 48세
48 = 2×21 + 6 = 42 + 6 ✓

✏ 추가 문제 (스스로 풀어보기)

현재 어머니의 나이는 딸의 나이보다 26살 많다. 5년 전에는 어머니 나이가 딸 나이의 5배였다면, 현재 어머니와 딸의 나이를 구하여라.

현재 딸 = x, 어머니 = x+26
5년 전: (x+26-5) = 5(x-5) → x+21 = 5x-25 → 4x=46 → x=11.5

→ 정수해가 아니면 문제 조건을 재확인! (이 문제는 설정에 따라 오류 가능)

딸: 12세, 어머니: 38세로 수정 시 → 5년 전 딸 7세, 어머니 33세 → 33≠35 (조건 불만족 확인 연습)
거리·속도·시간 문제
핵심 공식 하나: 거리 = 속도 × 시간

🔑 핵심 전략 — d=vt 표로 두 상황을 나란히

만나는 경우: 거리의 합 = 전체 거리

쫓아가는 경우: 이동 거리가 같아지는 순간

왕복: 갈 때 시간 + 올 때 시간 = 전체 시간

📌 대표 문제 — 서로 마주 보고 출발

A 도시와 B 도시는 120km 떨어져 있다. 두 버스가 동시에 서로를 향해 출발했다. 한 버스의 속도는 다른 버스보다 10km/h 빠르다. 두 버스가 48분 후에 만났다면, 각 버스의 속도를 구하여라.

🇫🇮 핀란드식 풀이

표
속도(km/h)시간(h)거리(km)
버스 Av4/54v/5
버스 Bv+104/54(v+10)/5
48분 = 48/60 = 4/5 시간. 단위를 시간으로 통일하는 것이 핵심!
방정식

두 버스 거리의 합 = 120km

4v/5 + 4(v+10)/5 = 120
4v + 4v + 40 = 600 → 8v = 560 → v = 70
답
버스 A: 70km/h, 버스 B: 80km/h
✓ 검산: (70+80) × 4/5 = 150 × 0.8 = 120km ✓

✏ 추가 문제

집에서 도서관까지 걸어가면 30분, 자전거를 타면 10분 걸린다. 자전거 속도가 걷는 속도보다 12km/h 빠를 때, 집에서 도서관까지의 거리는?

걷는 속도 = v km/h, 거리 = v × (1/2) = (v+12) × (1/6)
3v = v+12 → 2v = 12 → v = 6 km/h
거리 = 6 × 1/2 = 3km

📌 대표 문제 — 쫓아가기

동생이 먼저 분속 60m로 출발한 지 10분 후 형이 분속 100m로 같은 방향으로 출발했다. 형이 동생을 따라잡는 데 걸리는 시간은?

🇫🇮 핀란드식 풀이

핵심

형이 출발한 후 t분이 지났을 때 따라잡음

따라잡는 순간 = 두 사람의 이동 거리가 같아지는 순간!
표
속도(m/분)시간(분)거리(m)
동생60t + 1060(t+10)
형100t100t
방정식
60(t+10) = 100t → 60t + 600 = 100t → 40t = 600 → t = 15
답
형이 출발한 지 15분 후 따라잡음
✓ 동생 이동: 60×25=1500m, 형 이동: 100×15=1500m ✓

📌 대표 문제 — 왕복

올라갈 때 시속 3km, 내려올 때 시속 5km로 등산했더니 총 4시간이 걸렸다. 올라간 거리는?

🇫🇮 핀란드식 풀이

1

올라간 거리 = x km (내려온 거리도 x)

올라간 시간 + 내려온 시간 = 4
x/3 + x/5 = 4
왕복 문제: 거리는 같고, 시간의 합이 주어집니다!
2
5x/15 + 3x/15 = 4 → 8x = 60 → x = 7.5
올라간 거리: 7.5km
✓ 7.5/3 + 7.5/5 = 2.5 + 1.5 = 4시간 ✓
소금물 농도 문제
농도가 달라도 소금의 양은 보존된다!

🔑 핵심 원리 — 소금의 양은 항상 보존

소금의 양(g) = 농도(%) × 소금물의 양(g) ÷ 100

섞기 전 소금 합계 = 섞은 후 소금

물은 늘거나 줄지만, 소금은 절대 사라지지 않습니다!

📌 대표 문제

8% 소금물 300g에 몇 g의 물을 더 넣으면 5% 소금물이 되는가?

🇫🇮 핀란드식 풀이 — 소금 보존

시각화
8% 300g
소금 24g + 물 276g
물 추가
소금 24g + 물 (276+x)g = 5%
소금
처음 소금 = 300 × 8/100 = 24g
물을 아무리 넣어도 소금의 양(24g)은 변하지 않습니다!
방정식

물 x g을 넣은 후 총 소금물 = (300+x)g, 농도 5%

24 / (300 + x) = 5/100
2400 = 5(300+x) = 1500 + 5x → 5x = 900 → x = 180
답
물 180g을 더 넣는다
✓ 24 / 480 = 0.05 = 5% ✓

6% 소금물 200g과 10% 소금물 x g을 섞어 8% 소금물을 만들려 한다. x를 구하여라.

6%×200 + 10%×x = 8%×(200+x)
12 + 0.1x = 16 + 0.08x → 0.02x = 4 → x = 200

일의 양 문제 (작업량)
전체 일 = 1 로 놓고, 단위 시간당 일의 양으로 접근

🔑 핵심 전략 — 전체 일 = 1, 속도 = 1/걸리는시간

A가 혼자 n일 걸린다 → A의 하루 작업량 = 1/n

함께 작업: 각자의 하루 작업량을 더한다

📌 대표 문제

어떤 일을 혼자 하면 A는 12일, B는 18일이 걸린다. 먼저 A 혼자 4일 일하고, 나머지를 A, B 함께 마치면 총 며칠이 걸리는가?

🇫🇮 핀란드식 풀이 — 작업량 막대 모델

단위
A 하루 = 1/12
B 하루 = 1/18
함께 하루 = 1/12 + 1/18 = 5/36
남은 일

A 혼자 4일: 일의 양 = 4 × 1/12 = 1/3

남은 일 = 1 - 1/3 = 2/3
전체를 1로 보면 남은 비율을 분수로 쉽게 계산할 수 있습니다.
함께

함께 2/3 일 하는 데 걸리는 날 = t

5/36 × t = 2/3 → t = 2/3 × 36/5 = 72/15 = 24/5 = 4.8일
답
총 기간 = 4 + 4.8 = 8.8일
✓ 4×(1/12) + 4.8×(5/36) = 1/3 + 2/3 = 1 ✓
원가·정가·이익 문제
비율 관계를 한 변수로 연결하기

🔑 핵심 — 이익의 출발점은 항상 "원가"

  • 정가 = 원가 × (1 + 이익률)
  • 할인가 = 정가 × (1 - 할인율)
  • 이익 = 판매가 - 원가

📌 대표 문제

어떤 상품에 원가의 40% 이익을 붙여 정가를 매겼다가, 정가의 20%를 할인해서 팔았다. 이때 원가에 대한 실제 이익률은?

🇫🇮 핀란드식 풀이 — 원가를 100으로 설정

설정

원가 = 100 (편의상 100원으로 설정)

원가를 100으로 놓으면 퍼센트 계산이 덧셈/뺄셈으로 바뀝니다!
계산
정가 = 100 × 1.4 = 140원
판매가 = 140 × 0.8 = 112원
이익률
이익 = 112 - 100 = 12원
이익률 = 12/100 = 12%
원가 대비 이익이므로 분모는 항상 원가(100)입니다!

✏ 추가 문제

원가의 30% 이익을 붙여 정가를 매겼다. 정가에서 얼마를 할인해야 원가의 4% 이익이 남는가?

원가=100, 정가=130, 목표 판매가=104
할인액 = 130 - 104 = 26원
정가의 26/130 = 20% 할인
도형 넓이 응용

📌 대표 문제

가로가 세로보다 4cm 긴 직사각형이 있다. 가로를 3cm 줄이고 세로를 2cm 늘렸더니 넓이가 처음보다 6cm² 줄었다. 처음 직사각형의 넓이는?

🇫🇮 핀란드식 풀이

설정

세로 = x cm, 가로 = (x+4) cm

가로세로넓이
처음x+4xx(x+4)
변경 후x+1x+2(x+1)(x+2)
방정식
x(x+4) - (x+1)(x+2) = 6
x²+4x - (x²+3x+2) = 6
x + (-2) = 6 → x - 2 = 6 → x = 8
답

세로 8cm, 가로 12cm

처음 넓이 = 8 × 12 = 96cm²
✓ 변경 후: 9×10=90, 96-90=6 ✓
피타고라스 정리 응용
a²+b²=c² — 그림을 그리면 어디에 쓸지 보인다

📌 대표 문제

길이가 13cm인 사다리를 벽에 기대어 세웠다. 사다리의 아랫부분이 벽에서 5cm 떨어져 있다면, 사다리는 벽을 몇 cm 높이까지 닿는가? 만약 아랫부분을 7cm 더 밀면 위쪽은 몇 cm 내려오는가?

🇫🇮 핀란드식 풀이

1단계
h² + 5² = 13² → h² = 169 - 25 = 144 → h = 12cm
직각삼각형: 빗변(사다리)=13, 밑변(바닥)=5, 높이=h
2단계

아랫부분이 5+7=12cm로 이동

h'² + 12² = 13² → h'² = 169 - 144 = 25 → h' = 5cm
답
처음 높이: 12cm, 내려온 거리: 12 - 5 = 7cm
✓ 흥미로운 점: 아래로 7cm 밀었더니 위도 정확히 7cm 내려왔습니다. 우연이 아닌 3-5-12-13 피타고라스 수의 관계!

✏ 추가 문제

직각삼각형의 빗변이 25cm이고, 두 직각변의 합이 35cm일 때 두 직각변의 길이를 구하여라.

a+b=35, a²+b²=625
(a+b)²=a²+2ab+b²=1225 → 2ab=600 → ab=300
a와 b는 합35, 곱300 → t²-35t+300=0 → (t-15)(t-20)=0
두 변: 15cm와 20cm
연립방정식 응용
두 조건 → 두 방정식 → 두 미지수

🔑 핵심 전략 — 조건이 2개면 미지수도 2개

연립방정식: 미지수 2개, 조건(방정식) 2개 → 반드시 풀린다

풀이: ① 대입법 (한 식을 다른 식에 대입) ② 가감법 (두 식을 더하거나 빼기)

📌 대표 문제 — 학교 행사 입장료

어느 행사의 입장료는 어른과 청소년이 다르다. 어른 3명, 청소년 5명의 입장료 합계는 55,000원이고, 어른 2명, 청소년 4명의 합계는 40,000원이다. 어른과 청소년의 입장료를 각각 구하여라.

🇫🇮 핀란드식 풀이 — 가감법

설정

어른 입장료 = a원, 청소년 = b원

①  3a + 5b = 55,000
②  2a + 4b = 40,000
가감법

①×2 - ②×3 으로 a 소거:

6a+10b = 110,000
-) 6a+12b = 120,000
────────────
-2b = -10,000 → b = 5,000
계수를 같게 만들어 빼면 한 미지수가 사라집니다 — 가감법의 원리!
답

b=5,000 → ②에 대입: 2a + 20,000 = 40,000 → a = 10,000

어른: 10,000원, 청소년: 5,000원
✓ ①: 30,000+25,000=55,000 ✓
일차함수 응용 — 두 함수의 교점
교점 = 두 상황이 같아지는 순간

🔑 핵심 — 교점의 의미를 먼저 파악

두 직선의 교점 = 두 함수값이 같은 x값

실생활: "요금이 같아지는 사용량", "이익이 같아지는 판매량" 등

📌 대표 문제 — 두 통신사 요금제 비교

A 통신사: 기본료 15,000원 + 1분당 50원
B 통신사: 기본료 없음, 1분당 120원
몇 분 이상 사용해야 A 통신사가 유리한가?

🇫🇮 핀란드식 풀이

함수
A: y = 50x + 15,000
B: y = 120x
교점

요금이 같아지는 사용량: 50x + 15,000 = 120x

70x = 15,000 → x = 약 214.3분
답
215분 이상 사용하면 A 통신사가 유리
✓ 214분: A=25,700원, B=25,680원 (B가 약간 싸)
215분: A=25,750원, B=25,800원 (A가 싸) ✓
이차방정식 응용
x²이 등장하는 실생활 — 조건에 맞는 근 선택이 핵심

📌 대표 문제 — 토지 분할

넓이 200m²인 직사각형 토지가 있다. 가로를 4m 늘리고 세로를 3m 줄였더니 넓이가 처음과 같아졌다. 처음 직사각형의 가로와 세로는?

🇫🇮 핀란드식 풀이

설정

가로 = x m, 세로 = 200/x m

넓이=200이 주어졌으므로 세로를 x로 표현하지 않고 200/x로 쓰면 미지수가 1개!
방정식
(x+4)(200/x - 3) = 200
200 - 3x + 800/x - 12 = 200
-3x + 800/x = 12   (양변에 x 곱하기)
-3x² + 800 = 12x → 3x² + 12x - 800 = 0
근의 공식
x = (-12 ± √(144 + 9600)) / 6 = (-12 ± √9744) / 6
√9744 ≈ 98.7 → x ≈ (-12 + 98.7)/6 ≈ 14.45m
⚠ 음수 근은 "길이"가 음수가 될 수 없으므로 제외!
답
가로 ≈ 14.45m, 세로 ≈ 200/14.45 ≈ 13.84m
✓ (14.45+4)×(13.84-3) ≈ 18.45×10.84 ≈ 199.9 ≈ 200 ✓

✏ 추가 문제

어떤 자연수를 제곱하고 7을 더한 것이 그 수의 4배와 같다. 이 자연수를 구하여라.

x² + 7 = 4x → x² - 4x + 7 = 0
D = 16 - 28 = -12 < 0 → 실수해 없음

→ 문제 조건 재확인 필요! 올바른 문제: x² - 7 = 4x+5

x² - 4x - 12 = 0 → (x-6)(x+2) = 0 → x = 6 (자연수)
자연수: 6
확률 — 조건부 확률과 여사건
복잡해 보이는 확률도 경우의 수 표로 해결

🔑 핵심 전략 — 여사건 활용 "전부에서 안 되는 것 빼기"

P(A) = 1 - P(A가 일어나지 않는 경우)

"적어도 하나"를 구할 때 여사건이 훨씬 쉽습니다!

📌 대표 문제

남학생 4명, 여학생 3명 중에서 대표 2명을 뽑을 때, 적어도 한 명이 여학생일 확률은?

🇫🇮 핀란드식 풀이 — 여사건

전체
7명 중 2명: C(7,2) = 7×6/2 = 21가지
여사건

"적어도 한 명이 여학생"의 여사건 = "모두 남학생"

남학생 2명: C(4,2) = 6가지
직접 계산: (여1남1) + (여2) = 12+3 = 15가지 vs 여사건: 21-6=15 → 같은 답, 여사건이 훨씬 빠름!
답
P = 1 - 6/21 = 1 - 2/7 = 5/7

✏ 추가 문제

동전 3개를 동시에 던질 때 앞면이 적어도 1개 나올 확률은?

전체 = 2³ = 8가지
여사건: 모두 뒷면 = 1가지
P = 1 - 1/8 = 7/8
통계 — 평균·분산 응용
주어진 평균·분산에서 모르는 값 역추적

📌 대표 문제

5개의 수: 4, 7, a, 10, b가 있다. 이 자료의 평균이 8이고, 분산이 6일 때 a와 b를 구하여라. (단, a < b)

🇫🇮 핀란드식 풀이 — 연립방정식으로 역계산

평균
(4+7+a+10+b)/5 = 8 → a+b = 19   ···①
분산

편차: (4-8)²=16, (7-8)²=1, (a-8)², (10-8)²=4, (b-8)²

[16+1+(a-8)²+4+(b-8)²] / 5 = 6
(a-8)² + (b-8)² = 30 - 21 = 9   ···②
연립

①에서 b = 19-a → ②에 대입:

(a-8)² + (11-a)² = 9
a²-16a+64 + a²-22a+121 = 9
2a² - 38a + 176 = 0 → a² - 19a + 88 = 0
(a-8)(a-11) = 0 → a=8 또는 a=11
답

a < b이므로: a=8, b=11

a = 8, b = 11
✓ 평균: (4+7+8+10+11)/5=40/5=8 ✓
분산: (16+1+0+4+9)/5=30/5=6 ✓

🚗 거리·속도·시간

  • d=vt 표 먼저 → 관계 파악
  • 단위 통일 (시간 vs 분 주의!)
  • 만남: 거리 합=전체 / 쫓기: 거리 같음
  • 왕복: 거리 같음, 시간 합=전체

🧪 농도·일의 양

  • 소금의 양 = 농도 × 소금물 / 100
  • 섞어도 소금 총량은 보존
  • 일의 양: 전체=1, 속도=1/일수

🏷 원가·이익·할인

  • 원가를 100으로 놓으면 계산 단순화
  • 정가 = 원가 × (1 + 이익률)
  • 이익률 분모는 항상 원가

📐 도형·이차방정식

  • 반드시 그림 먼저, 변수 표기
  • 이차방정식 → 두 근 모두 구하고 조건 확인
  • 길이·개수 → 양수 근만 유효

🎲 확률·통계

  • "적어도 하나" → 여사건 (1 - P(반대))
  • 평균·분산 역계산 → 연립방정식
  • 검산: 구한 값을 조건에 대입

🇫🇮 핀란드식 문제 풀이 3원칙
