# 🔢 수학왕국 드릴형

**아르모니아 프로젝트** — 초등·중등 수학 반복 드릴 앱

🌐 **라이브**: https://choisuka.github.io/math_game/

---

## 내용

| 대상 | 단원 |
|---|---|
| 초등 1~6학년 | 핀란드식 초등수학 전 과정 |
| 중학교 1학년 | 정수·유리수, 문자와 식, 방정식 |
| 중학교 2학년 | 함수, 연립방정식, 부등식 |
| 중학교 3학년 | 이차방정식, 이차함수, 확률 |

## 학습 방식

- 🇫🇮 **핀란드식** 반복 드릴 — 속도·정확도 훈련
- 10문항 게임 형식, 별점 결과
- 게임화: 유물 획득, 항해 일지, 나침반 힌트
